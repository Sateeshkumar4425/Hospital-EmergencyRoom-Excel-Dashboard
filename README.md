# 🏥 Hospital Emergency Room Dashboard (Excel Project)

## 📊 Project Overview
This project analyzes hospital emergency room data using Microsoft Excel to track patient flow, wait times, and service efficiency.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## 📈 Key Metrics
- Total Patients: 513
- Average Wait Time: 36.32 minutes
- Patient Satisfaction Score: 4.96
- Admission Rate: ~50%

## 📊 Dashboard Features
- Monthly patient analysis
- Department-wise patient distribution
- Age group analysis
- Gender-wise insights
- Admission vs non-admission comparison
- Patient wait time tracking

## 📷 Dashboard Preview
![Dashboard](Hospital_Emergency_Room_DB.png)

## 💡 Key Insights
- Majority of patients come through General Practice
- Nearly equal admission and non-admission rates
- Some delays observed in patient processing

## 🚀 Skills Demonstrated
- Advanced Excel
- Data Analysis
- Data Visualization
- Business Insights

## 📌 Conclusion
This dashboard helps improve hospital decision-making by providing clear insights into patient trends and operational efficiency.
